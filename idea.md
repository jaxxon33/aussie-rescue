Aussie 4WD Rescue Website

PHP/Google Maps/Mobile/JavaScript/HTML5

A mobile and desktop website that allows users to see each other's locations in real time on a map, updated continuously from gps. 

Must log on to see vehicle locations using email address and password. Must register first. Profile fields include username, name, status, phone number, CB channel, vehicle type, modifications, URL, Rescue Rig (checkbox), location (lat, lon)

Users can call for help which turns their location marker red. Other users can select the red marker then select 'attending' to indicate they are on the way. Both markers will then flash. Other vehicles can also indicate they are 'attending'.

Color Codes
-----------------
My vehicle location = steady blue
Other Vehicle = steady green
Other vehicle attending = flashing green
Stranded vehicle =  steady red
Stranded vehicle with vehicle attending = flashing red
